# Just Joke

一个基于 Vue.js 开发的平台。

## 功能特性

- 与 AI 对话
- 低代码平台
- 简洁优雅的界面设计

## 技术栈

- Vue 3
- Vue Router
- Axios
- Font Awesome 图标
- Normalize.css

## 快速开始

1. 克隆项目:

```bash
git clone <repository-url>
cd justJoke
```

2. 安装依赖:

```bash
npm install
```

3. 启动开发服务器:

```bash 
npm run serve
```

4. 打开浏览器访问 http://localhost:8080

## 项目结构

```
src/
  ├── assets/        # 静态资源
  ├── components/    # 组件
  ├── router/        # 路由配置
  ├── services/     # API 服务
  ├── App.vue       # 根组件
  └── main.js       # 入口文件
```

## 构建部署

构建生产环境版本:

```bash
npm run build
```
